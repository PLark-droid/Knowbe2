# Agent Performance Metrics & SLA

## 共通メトリクス

| メトリクス | 説明 | 計測方法 |
|-----------|------|---------|
| execution_time | 実行時間 | start~end timestamp |
| success_rate | 成功率 | completed / total |
| retry_count | リトライ回数 | 累積カウント |
| error_rate | エラー率 | failed / total |
